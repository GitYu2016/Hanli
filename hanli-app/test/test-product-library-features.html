产品库功能测试

测试产品库的新功能：显示字段、排序、自动刷新、分页

1. 显示字段测试

验证产品库显示以下字段：

  • 产品标题 (goodsCat3) - 最大50字符，超出显示省略号
  • 昨日销量 (yesterdaySales) - 格式：数字 + "件"
  • 价格增长百分比 (priceGrowthPercent) - 格式：+X.X% 或 -X.X%，正数绿色，负数红色
  • 采集日期 (collectTime) - 格式：YYYY-MM-DD HH:mm

2. 排序功能测试

验证以下排序功能：

  • 默认按采集时间倒序（最近采集的在最上面）
  • 点击列标题可切换升序/降序
  • 表头显示排序指示器（↑↓↕）
  • 支持多列排序

3. 自动刷新测试

验证以下刷新功能：

  • 进入产品库页面时立即刷新一次
  • 每5秒自动刷新一次
  • 离开页面时停止自动刷新
  • 显示刷新状态和最后更新时间

4. 分页功能测试

验证以下分页功能：

  • 每页显示100个产品（默认）
  • 显示当前页/总页数信息
  • 支持翻页操作
  • 显示总产品数
