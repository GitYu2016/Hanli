🪟 窗口控制按钮位置调整测试

当前设置

📋 当前配置

mainWindow = new BrowserWindow({ titleBarStyle: 'hiddenInset', titleBarOverlay: { color: 'transparent', symbolColor: '#000000', height: 44, // 增加高度来向下移动按钮 y: 40 // 向下偏移40px } });

说明：当前设置尝试将窗口控制按钮向下移动40px

调整说明

🎯 调整目标

将macOS窗口左上角的三个控制按钮（关闭、最小化、最大化）向下移动，使其不与顶部内容重叠。

期望效果：控制按钮应该向下移动，给顶部导航栏留出更多空间

配置参数说明

titleBarOverlay: { color: 'transparent', // 覆盖区域颜色 symbolColor: '#000000', // 控制按钮颜色 height: 44, // 覆盖区域高度 y: 40 // 垂直偏移量（向下移动） }
  • height: 控制覆盖区域的高度，影响按钮的点击区域
  • y: 控制按钮的垂直位置，正值向下移动
  • titleBarStyle: 'hiddenInset': 使用macOS的默认样式

可能的解决方案

🔧 如果按钮仍然没有下移

方案1: 增加偏移量

titleBarOverlay: { y: 60 // 尝试更大的偏移量 }

方案2: 调整titleBarStyle

titleBarStyle: 'hidden', // 完全隐藏标题栏 titleBarOverlay: { y: 20 }

方案3: 使用不同的配置组合

titleBarStyle: 'hiddenInset', titleBarOverlay: { color: 'transparent', symbolColor: '#000000', height: 50, y: 50 }

测试步骤

  1. 保存main.js文件
  2. 重启Electron应用
  3. 观察窗口左上角的三个控制按钮位置
  4. 如果位置仍然不正确，尝试调整y值
  5. 每次调整后都需要重启应用才能看到效果

注意事项

  • macOS的窗口控制按钮位置调整可能受到系统版本影响
  • 不同的Electron版本可能有不同的行为
  • 如果使用自定义标题栏，可能需要完全重新实现
  • 调整后需要重启应用才能看到效果
