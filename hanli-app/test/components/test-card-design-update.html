5. 倒角变量说明

CSS 变量定义
:root {
    --radius-small: 4px;      /* 小倒角 */
    --radius-medium: 6px;     /* 中等倒角（标准） */
    --radius-large: 12px;     /* 大倒角 - 卡片专用 */
    --radius-card: 12px;      /* 卡片专用大倒角 */
}
