💻 使用示例

/* 基础用法 */ .button { background-color: var(--color-primary); color: var(--color-primary-foreground); border: 1px solid var(--color-border); } .button:hover { background-color: var(--color-primary-hover); } /* 文本颜色 */ .title { color: var(--color-text-primary); } .description { color: var(--color-text-secondary); } /* 卡片样式 */ .card { background-color: var(--color-card-background); border: 1px solid var(--color-border); box-shadow: var(--shadow-hover); }

🌓 主题切换

所有颜色变量都支持亮色和暗色主题的自动切换。主题切换时，所有使用CSS变量的元素都会自动更新颜色。

/* 主题切换 */ body.theme-light { /* 亮色主题变量 */ } body.theme-dark { /* 暗色主题变量 */ }
